# machine-learning
Repository to hold all my machine learning projects and environments

## Setup
- [Install Conda](https://docs.conda.io/projects/conda/en/latest/user-guide/install/index.html#)
- [Create an Environment](https://docs.conda.io/projects/conda/en/latest/user-guide/tasks/manage-environments.html#create-env-file-manually)
    - ```conda create -f environment.yml```
- Run environment
    - ```conda activate machine-learning```
